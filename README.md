# HTV32BloatwareRemover
HTC 10 HTV32のプリインストールアプリ(Bloatware)を一括アンインストールするバッチです。

## 準備
【PC側】
adbコマンドを実行できるようにしておく
USB Type-CケーブルでHTC 10 HTV32と接続しておく

【端末側】
開発者オプションを有効化、USBデバッグをオンにしておく

※ プリインストールのキーボードも削除する為、先にキーボードアプリをインストールしておく、
　 もしくは処理後設定でキーボードを変更・有効化してください。

## 手順
端末とPCを接続し、バッチファイルを実行するだけです。
